# מערכת שידוכים – שאלון היכרות + ניהול מועמדים

מערכת CRM פנימית ושאלון היכרות, בנויה ב-Next.js ו-Tailwind CSS, מותאמת 100% למובייל, בעברית מלאה.

## מה יש במערכת

**צד המועמד (השאלון):**
- מסך פתיחה עם הלוגו הגדול, כותרת ובחירת מסלול (בחור / בחורה).
- שאלון מלא עם כל השדות (פרטים אישיים, תמונה, שאלות פתוחות, אנשי קשר). כל השדות חובה.
- לאחר השליחה לא ניתן לערוך. מסך סיום עם הודעת תודה.

**מערכת הניהול (כניסה דרך "כניסת צוות ניהול" במסך הפתיחה):**
- **מנהלת:** רואה את כל המידע, מוחקת כפילויות, משייכת מועמדים לנציגים, עורכת טופסי מועמדים ועורכת את שאלות השאלון.
- **נציג:** רואה עמודה משלו עם המועמדים המשויכים אליו (שם הנציג ושם המוסד בראש העמודה).
- כרטיס מקוצר (תמונה, שם, גיל, מגדר, נציג) וכרטיס מורחב (הטופס המלא).
- **מידע רגיש:** גלוי רק לנציג ולמנהלת.
- **ייצוא:** העתקה ללוח והורדת PDF – נקיים, ללא מידע רגיש וללא מספרי טלפון לבירורים.
- **פעולות מתקדמות:** הוספת מועמד, משימות עם תאריך יעד, והתאמות (Match) עם עדכון "הצעד הבא" וכפתור "יצירת קשר" בוואטסאפ.

> הנתונים נשמרים בדפדפן (localStorage) כדי שהמערכת תעבוד מיד כאתר דמה, ללא צורך בשרת או במסד נתונים.

## העלאה לאוויר (Vercel) – שלב אחר שלב

1. להיכנס ל-https://vercel.com ולהתחבר עם חשבון GitHub.
2. ללחוץ **Add New… → Project**.
3. לבחור את המאגר הזה (breslev-shidduchim) וללחוץ **Import**.
4. להשאיר את כל ההגדרות כפי שהן וללחוץ **Deploy**.
5. תוך כדקה מתקבל קישור חי לאתר (למשל `https://breslev-shidduchim.vercel.app`).

## הרצה מקומית (למפתחים)

```bash
npm install
npm run dev
```

הדפדפן: http://localhost:3000
